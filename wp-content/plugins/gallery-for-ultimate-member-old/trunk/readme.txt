== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gallery-for-ultimate-member` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Ultimate Member -> UM Gallery screen to configure the plugin


== Frequently Asked Questions ==

= What plugins do I need? =

This plugin requires [Ultimate Member](https://wordpress.org/plugins/ultimate-member/)

= How many photos can my users upload? =

Users are able to upload an unlimited amount of photos.

= Is it easy to upgrade to Pro Version? =

Yes. You can download the pro version from [Gallery for Ultimate Member](https://suiteplugins.com/downloads/gallery-for-ultimate-members/)

= Where can I place the translation file? =

The files can be placed in **wp-content/languages/plugins/** with the file name **gallery-for-ultimate-member-en_US.mo**.

= Does the media files get added to WordPress Media? =

No. Files are added to a custom folder in the wp-content folder and are not saved as a post type but in a custom table.

= Will there be video uploads? =

There aren't any immediate plans to introduce video uploads such as MP4 at this stage.
